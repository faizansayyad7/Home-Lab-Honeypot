# 🛡 Home Lab Honeypot Dashboard

A web-based cybersecurity project built using Flask and Python that simulates honeypot-style monitoring in a controlled environment. The application records interaction events, analyzes repeated activity patterns, identifies suspicious IP behavior, and visualizes security insights through an interactive dashboard.

---

## 🚀 Features

✅ Activity event logging

✅ Source IP tracking

✅ Timestamp monitoring

✅ Security alerts

✅ Suspicious activity detection

✅ Dashboard statistics

* Total Events
* Unique IPs
* Suspicious IPs

✅ Recent activity monitoring

✅ RGB Cyber-Themed Dashboard

---
